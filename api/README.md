# Temporary Email Service for Netlify

A complete temporary email service that can be deployed on Netlify with serverless functions.

## Features

✅ **Instant Email Creation** - Generate temporary emails in seconds
✅ **Real Email Reception** - Receive actual emails (requires SMTP setup)
✅ **Web Dashboard** - Beautiful, responsive dashboard
✅ **REST API** - Full API for integration
✅ **Email Forwarding** - Send emails from your temp address
✅ **Auto-expiry** - Emails automatically delete after set time
✅ **Password Protection** - Optional password for email access
✅ **QR Code Generation** - Easy mobile access
✅ **Security** - Rate limiting, input sanitization, CORS
